INCOMPLETE FEATURES
===================

TILE (classes/maps.py)

CREATURE STATS (classes/stats.py)
- Stats class with base/derived/mods/active layers implemented
- Derived stat formulas defined but most not yet consumed by gameplay mechanics
- Opposing stat contest system defined but no combat/social system calls it yet
- Stat point allocation on level-up implemented but no UI to spend points

ITEMS / INVENTORY (classes/inventory.py)
- damage, defense: defined on Weapon/Ammunition but no combat system to consume them
- health: defined but no consume/use mechanic
- poison: defined but no poison mechanic
- durability / durability_current: defined but never decremented or checked
- value: defined but no economy system
- weight: used by encumbrance (carried_weight vs CARRY_WEIGHT) — checked on pickup
- equippable, slots, slot_count: equip/unequip logic now on Creature
- consumable, stackable, quantity: Stackable.add() and coalesce() implemented but never called
- ITEMS dict (data/db.py): populated from DB at load but never accessed during gameplay
- STRUCTURES dict (data/db.py): populated from DB but never accessed (instances render via WorldObject)
- inventoriable: checked by pickup() but no UI or AI triggers pickup yet
- pickup/drop implemented on Creature but no runtime caller (no input binding, no AI action)
- No items or structures are spawned at runtime — no loot system

CREATURE METHODS (classes/creature.py)
- transfer_item(): fully implemented but never called anywhere
- pickup/drop: implemented with encumbrance + inventoriable checks, not called yet

CREATURE TRAITS (classes/creature.py)
- sex: male/female per-creature (randomly assigned if not set), no gameplay effect yet
- prudishness: 0.0–1.0 species default with per-creature override, no gameplay effect yet
- age: days on the map (0 = newborn/hatched), age_class property (young/adult/old)
- age thresholds currently hardcoded — will move to species config

CREATURE TEMPLATES (data/db.py, editor/creatures_tab.py)
- creatures + creature_stats tables for named NPC definitions
- Species template + stat/trait overrides (level, sex, age, prudishness, behavior, items)
- Loaded into CREATURES dict but no spawning system consumes them yet

MATING / PAIRING (not yet implemented)
- Pairings produce eggs (Item) placed on the ground
- Eggs can be: scavenged by other creatures, hatch in place, or picked up
- Cross-species pairing → "SpeciesAxSpeciesB" abomination offspring
- Abominations shunned by most creatures (large negative disposition penalty)
- Trait inheritance: offspring stats derived from parents via combination
- Hatched creatures start at age 0

DIALOGUE SYSTEM (classes/creature.py, data/db.py, editor/dialogue_tab.py)
- dialogue table: conversation trees with parent-child nodes (parent_id=NULL for roots)
- Filters: species, creature_key, char_conditions, world_conditions, quest_conditions
- start_conversation(): finds matching roots, enters conversation state
- advance_dialogue(): selects node, applies effects, returns filtered children
- Effects: give_item, take_item, sentiment shift, behavior trigger
- Editor tab: tree view + node edit form with all fields
- World/quest condition checking not yet wired (needs quest/world state system)
- NPC-to-NPC conversations supported (no limits yet)

NPC BEHAVIOR (classes/creature.py)
- NPC subclass removed; all creatures use Creature with behavior modules
- Only RandomWanderBehavior exists — no pathfinding, aggro, patrol, or scheduling
- dialogue attribute exists as placeholder (None) — no dialogue system yet

COMBAT
- melee_attack() implemented: stamina cost, sight/ambush, dodge contest, armor resist,
  damage (STR + weapon + lucky STR), crit, stagger, HP regen reset, betrayal detection
- ranged_attack() implemented: distance accuracy, ammo consumption, recoverable ammo
  (arrows/rocks land on target tile on miss), non-recoverable (bullets consumed)
- grapple() implemented: max(STR,AGL) vs max(STR,AGL-1), stamina cost, dominance
- enter_block_stance/exit_block_stance: requires item in hand, +2 BLOCK while active
- cast_spell() implemented: mana/stamina cost, requirements, dodge contest (if dodgeable),
  magic resist (hostile only), 4 effect types (damage/heal/buff/debuff), timed buff expiry,
  secondary resist (e.g. poison from spell), self-targeted spells
- Spell definitions in spells table, creature_spells + species_spells for known spells
- Attack behaviors (animation bindings) not wired to attack methods yet
- die() implemented: drops inventory + equipment on tile, clears mods, plays death anim
- is_alive property checks HP > 0
- Trap trigger on tile entry: detection vs trap_dc, damage on fail, trap consumed

SOCIAL
- intimidate() implemented: d20 INTIMIDATION vs FEAR_RESIST, relationship recording
- deceive() implemented: d20 DECEPTION vs DETECTION, trust damage on failure
- steal() implemented: stealth/deception contest, item transfer, caught = trust damage
- share_rumor() implemented: CHR-scaled probability, confidence from relationship
- propose_trade() implemented: utility scoring, sentiment modifier, persuasion bonus,
  fair/unfair detection, item swap on accept, walk-away penalty on reject
- bribe() implemented: value threshold scales with negative sentiment (hostile = bigger bribe needed)
- No seduce yet (needs species compatibility + pairing system)

EQUIPMENT
- equip/unequip implemented with stat mod passthrough and stat requirement checks
- render_on_creature field on Weapon/Wearable has no rendering path
- action_word defined on items but no description-building system consumes it yet
- requirements defined on items (JSON stat→min) — checked during equip()

ECONOMY
- No shop, trade, or currency system
- Item value field unused

SIGHT / STEALTH
- can_see() implemented: SIGHT_RANGE - target's STEALTH vs Manhattan distance
- Used by melee/ranged attack (ambush detection) and social actions (range check)
- Not yet used by rendering or AI observation space

CONSUMABLES
- use_item() implemented: applies buffs as timed stat mods, decrements quantity, removes at 0
- Duration expiry implemented: timed tick auto-removes mods after duration seconds
- Duration=0 means permanent (no expiry)

MOVEMENT VARIANTS
- run() implemented: stamina cost 3, same movement but flagged as running
- sneak() implemented: stamina cost 1, stealth movement
- flee() implemented: move away from threat, stamina cost
- Run/sneak speed differences not yet applied to tick intervals

UTILITY
- search_tile() implemented: reveals tile inventory, hidden_dc check vs DETECTION
- guard()/stop_guard() implemented: +3 DETECTION while guarding, stamina drain
- wait() implemented: skip turn, recover stamina (STAM_REGEN amount)
- follow() implemented: move one tile toward target (simple, no pathfinding)
- call_backup() implemented: broadcast within HEARING_RANGE, filters by positive sentiment
- sleep()/wake() implemented: full stamina+mana restore, -5 DETECTION while asleep
- set_trap() implemented: place trap item on tile, DC = base + CRAFT_QUALITY
- Sleep deprivation implemented: 4-tier stacking debuffs (mild/exhaustion/severe/collapse)
  sleep_debt tracks days, add_sleep_debt() applies/updates fatigue mods, sleep clears all
- Trap trigger implemented in move(): detection vs trap_dc on tile entry
- No use/consume action or effect system

DURABILITY
- durability_max and durability_current defined on Weapon/Wearable
- Never checked or decremented
